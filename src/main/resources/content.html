Branching

Create a local Branch

```shell git branch release1 git branch release2 ```
Switch to a branch
```shell git checkout release1 ``` After committing changes, if the branch is to be pushed to the remote repository ```shell git push origin release1 ``` Add the files you have worked upon This is the directory which I made the changes `git add .` (it will take care of ignore files)
